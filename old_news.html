Old News


6 May 2006

Version 1.7.1

New features:
  • Slider control
  • run() convenience function
  • Geometry.rects_intersect() function
Enhancements:
  • Filenames on command line opened at application startup
  • BlobEdit example extended to demonstrate update_rect usage
Bug fixes:
  • Disabled a stray debugging statement in Cursor.__init__
  • Fixed package name in setup.py
1 April 2006

Version 1.7

New features:
  • GL.DisplayList class for managing OpenGL display lists in a similar way to the GL.Texture class inroduced in 1.6.
  • Facilites for displaying a chosen cursor when the mouse is over a View.
  • Facilities for locating resources associated with an application.
Bug fixes:
  • Cocoa: Getting the extent property of a ScrollableView did not work.
  • Cocoa: Worked around a PyObjC bug that was causing Tasks to leak memory.
  • Cocoa: Request New File dialog with a file type having a Macintosh type code did not work.
Plus a number of other modifications, see CHANGES.

14 June 2004
Version 1.3
For the first time, PyGUI has two simultaneously working implementations covering different platforms: MacOS X using Carbon, and Linux using Gtk. The Gtk version should also be usable on other Unix/X11 platforms. and, at a pinch, on Windows, although a native Windows version would be preferable.
