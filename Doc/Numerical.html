module Numerical

The Numerical module provides some facilities for interfacing with NumPy.

Functions

image_from_ndarray(array, format, size = None)
Creates an Image from a NumPy array of type uint8. The format may be 'RGB' or 'RGBA'. If a size is specified, it should be a tuple (width, height), and the array will be implicitly reshaped to match the specified size and format. Otherwise the array must have 3 dimensions, the size is inferred from the first two dimensions of the array, and the third dimension must match the number of bytes required by the format.

If possible, the returned Image will share data with the original array, but this is not guaranteed. The effect of modifying the array after creating the image is undefined.
